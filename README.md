# amr_phenotype_cnn
Developing a CNN to outperform rules-based AMR phenotype prediction for novel variants of mutation driven resistance in pathogens like TB, NG, other pathogens

# The Hacking Plan

Step 1: Get a models (“MD-CNN” and “SD-CNN”) running via command line

Step 2: Develop an application (web or shiny app) for users to upload and get a prediction

Step 3: Release to beta testers and get feedback

Step 4: Compare with AMRrules and other tools

Step 5: Add more loci and data from Allthebacteria and re-train (“Curation”)

Step 6: Automate curation

Step 7: Adapt for other pathogens?

Step 8: Write a paper and publish
